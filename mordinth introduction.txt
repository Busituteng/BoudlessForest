Adventure Mobs Utility

boundlessforest



This is a small doujin adventure mod about the "bat happy meme".

If you want to know about this meme, please see the wiki page on the left side of this page (Chinese).

This mod added a dimension, a mob, a weapon and some items.

You can build the portal with oak leaves and start it with the ignitor in this mod to get into the dimension.

As it is a meme mod, so you don't need to be strict with items in the mod.

The mod has linkage with Torcherino Mod.
